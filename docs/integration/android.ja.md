# Android NDK 統合

詳細な手順は GitHub の [`integration/android/README.md`](https://github.com/ykus4/kagura/blob/main/integration/android/README.md) (英語) を参照してください。

> ⚠️ このページの日本語訳は今後追加予定です。最新かつ完全な手順は上記英語版にあります。

## 概要

- `apply from: "kagura/integration/android/kagura.gradle"` を `build.gradle` に追加
- CMake ベースのプロジェクトには `integration/android/kagura.cmake` ヘルパー
- ABI: arm64-v8a / armeabi-v7a / x86_64 対応

詳細は英語版 README を、関連項目は [統合](index.md) を参照してください。
